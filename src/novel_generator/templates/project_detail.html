What a run produces

A healthy run should finish every requested chapter before the manuscript export step begins.

  1. 1
    Outline the full book

    Create a chapter-by-chapter structure sized for {{ run_values.requested_chapters }} requested chapter{{ "" if run_values.requested_chapters == 1 else "s" }}.

  2. 2
    Draft every chapter

    Each chapter gets a beat plan, drafted prose, and a checkpoint in the database.

  3. 3
    Save continuity summaries

    Those summaries feed later chapters so reruns and regeneration stay coherent.

  4. 4
    Export only after drafting is complete

    If the worker cannot finish every chapter, the run should fail instead of claiming success.
